Welcome to HTML!

HTML stands for HyperText Markup Language, and it is a programming language used to describe the contents of webpages and what they look like. Ordinarily, a webpage is saved like any other file, but has the extension '.htm' or '.html'. With proper tools, such as JavaScript and CSS, a programmer has powerful methods at their disposal to code an HTML document and give it attractive features.
